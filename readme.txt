A simple plugin that integrates Traklight into WordPress, making use of the login api

== Description ==

This plugin inserts the Traklight iframe into your Wordpress install. It will use the currently logged-in user's email to authentiate with Traklight so your users can have a seemless, single login experience.

== Installation ==

1.) Install and activate plugin
2.) Insert shortcode `wp-traklight` into a page
